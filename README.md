Topic: Hand written digits recognition on Android.

[Slides](https://nbviewer.jupyter.org/github/blan4/ML_Workshop_2017-12-17/blob/master/slides.ipynb)

Used technologies:
- Kotlin for Android
- Android Anko 
- tensorflow-android
- Python for model
- Keras
- Tensorflow
- MNIST dataset to train model
